 Market Basket Analysis Using Apriori Algorithm

 Overview
This project uses the Apriori Algorithm to perform Market Basket Analysis on the Online Retail Dataset. The goal is to identify products that are frequently purchased together and generate association rules for business recommendations.

 Tools Used
- Python
- Pandas
- MLxtend
- Matplotlib
- Google Colab

 Dataset
- Online Retail Dataset
- 541,909 records
- UK-based online retailer

 Results
- Generated frequent itemsets using Apriori
- Created association rules using Support, Confidence, and Lift
- Identified strong product associations for cross-selling opportunities
